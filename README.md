FDProductManagement
API Rest para gerenciamento de produtos.

Configuração
Mudar connectionString nos arquivos appsettings.json e rodar script na pasta Scripts. Lembre-se de estar com o sdk atualizado.

ASP.NET WebApi Core

AutoMapper
